# Kanban Board - Frontend

This is the frontend application for the Kanban Board project. It provides a modern, interactive user interface for managing boards, lists, and cards.

# Requirements

- [Node.js](https://nodejs.org/)

## Setup

```bash
npm install # Install project dependencies
```

## Usage

### Development

```bash
npm run dev # Starts development server at http://localhost:5173
```

### Production

```bash
npm run build # Generates production build at ./dist
npm run preview # Start preview server at http://localhost:4173 (Requires production build)
```

## Linting & Formatting

```bash
npm run lint # Fix linting errors
npm run lint:check # Check for linting errors
npm run format # Format code
npm run format:check # Check for formatting errors
```

## Type Checking

```bash
npm run typecheck # Check for type errors
```

## Testing

```bash
npm run test:unit # Run unit tests
npm run test:e2e # Run end-to-end tests
npm run coverage # Run unit tests with coverage
```
